# MacOS美化终端 Terminal —— oh-my-zsh

# 准备

  • powerline字体
  • oh-my-zsh软件
  • oneDark主题包

# 安装

  • 安装oh-my-zsh
git clone https://github.com/robbyrussell/oh-my-zsh.git
cd tools
./install.sh
  • 安装主题
git clone https://github.com/mbadolato/iTerm2-Color-Schemes.git
/teminal/OneHalfDark.terminal

在终端中导入自己喜欢的主题

  • 安装powerline字体
git clone https://github.com/powerline/fonts.git --depth=1
cd fonts
./install.sh

字体安装完成后就可以在终端-偏好设置中设置

字体选择XXX for poweline（可根据个人喜好选择-某些字体会导致乱码）

个人喜欢用Fira code

  • 命令行高亮插件
cd ~/.oh-my-zsh/custom/plugins/
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
  • 配置信息
vi ～/.zshrc

ZSH_THEME="agnoster"  
#使用 agnoster 主题，很好很强大
DEFAULT_USER="你的用户名"   
#就是xxx@XXX @前面的用户名
#增加这一项，可以隐藏掉路径前面那串用户名
#自己按需把要用的 plugin 写上
plugins=(
git
brew
node
npm
zsh-syntax-highlighting #必须最后一个
)
